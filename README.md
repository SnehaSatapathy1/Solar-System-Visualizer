# Solar System Visualizer

A web-based 3D visualization of the solar system with real astronomical data.

## Features
- 3D visualization of planets and moons
- Real orbital positions based on user-input dates
- Interactive controls
- Responsive design

## Tech Stack
- Three.js for 3D rendering
- JavaScript/ES6 modules

## Getting Started

### Prerequisites
- Node.js (v14 or higher)

### Installation

```bash
git clone https://github.com/SnehaSatapathy1/Solar-System-Visualizer.git
cd Solar-System-Visualizer
npm install
```
### Start the development server
npm start
